Dataset & Features

This project uses customer account data with features such as credit score, country, gender, age, tenure, account balance, product count, card ownership, active status, and estimated salary.

  • Key numeric: credit_score, acc_balance, estimated_salary
  • Key categorical: country, gender
  • Binary flags: has_card, is_active
Inputs Required

You'll be asked to provide the following values on the prediction form. Example inputs are provided as placeholders.

  • credit_score: numeric (e.g., 650)
  • country: France / Spain / Germany / unknown
  • gender: Male / Female
  • age: numeric (e.g., 34)
  • tenure: years with bank (e.g., 3)
  • acc_balance: account balance (e.g., 120000)
  • prod_count: number of products (e.g., 2)
  • has_card / is_active: 0 or 1
  • estimated_salary: numeric (e.g., 75000)
